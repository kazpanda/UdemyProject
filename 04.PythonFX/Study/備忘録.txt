TA-LIBのインストール方法

ルートにダウンロードした、TA_Lib-0.4.19-cp37-cp37m-win_amd64.whlを保存
下記のコマンドを実行
# pip install TA_Lib-0.4.19-cp37-cp37m-win_amd64.whl

VSCodeでインテリセンスが効かないとき
下記を追加する
~~~
settings.json
{
    "python.analysis.extraPaths": [
        "DoNotTouchThisFolder"
    ], 
}
~~~

omitempty出来ない対応
~~~
set PYTHONUTF8=1
pip install omitempty
~~~
